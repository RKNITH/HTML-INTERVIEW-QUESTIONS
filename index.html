Top 30 Commonly Asked HTML Interview Questions

Easy

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages and web applications. HTML describes the structure of a web page using markup. HTML elements represent different parts of the page, such as headings, paragraphs, links, and more.

2. How do you create a hyperlink in HTML?

You create a hyperlink using the <a> tag. The href attribute specifies the destination URL.

<a href="https://example.com">Visit Example</a>

3. How do you include CSS in an HTML document?

CSS can be included in an HTML document in three ways: inline styles, internal styles using the <style> tag, or external stylesheets using the <link> tag.

4. How do you add an image to an HTML page?

You can add an image using the <img> tag with the src attribute to specify the image source.

<img src="image.jpg" alt="Description of image">

5. How do you create a list in HTML?

HTML provides two types of lists: ordered lists (<ol>) and unordered lists (<ul>). List items are added using the <li> tag.

Medium

6. What is the purpose of the <meta> tag in HTML?

The <meta> tag provides metadata about the HTML document, such as charset, description, and keywords. It helps with SEO and rendering of the page.

7. What is the difference between <div> and <span> tags?

The <div> tag is a block-level element, while the <span> tag is an inline element. <div> is used for grouping larger content, and <span> for smaller text within a line.

8. What does the <form> tag do?

The <form> tag is used to create a form for user input. It typically includes input fields, checkboxes, radio buttons, and a submit button.

9. How do you create a table in HTML?

A table is created using the <table> tag, with rows created using <tr>, and data cells within each row using <td>.

10. What are semantic HTML elements?

Semantic HTML elements clearly describe their meaning in a human- and machine-readable way, like <article>, <section>, and <footer>, as opposed to non-semantic elements like <div> or <span>.

Advanced

11. What is the purpose of the <!DOCTYPE html> declaration?

The <!DOCTYPE> declaration informs the browser about the version of HTML being used, ensuring proper rendering of the page.

12. What are HTML5 APIs and how do they work?

HTML5 APIs are a set of web technologies that allow for more dynamic interactions, including Web Storage, Geolocation, and Canvas APIs, enabling features like offline storage, location-based services, and drawing graphics.

13. How do you handle responsive design in HTML?

Responsive design in HTML is achieved using CSS media queries and viewport settings to make web pages adapt to different screen sizes and devices.

14. What is the difference between the id and class attributes, and when should each be used?

The id attribute is used for uniquely identifying a single element, while the class attribute can be used for multiple elements. id should be used for one-time identification, whereas class is used for applying styles or behaviors to a group of elements.

15. How do you implement HTML5 local storage and session storage?

Local storage and session storage are part of HTML5â€™s Web Storage API. Local storage persists data across sessions until explicitly deleted, while session storage stores data for the duration of the page session. You can use the localStorage.setItem() and sessionStorage.setItem() methods to store data.

16. What are custom data attributes and how are they used in HTML?

Custom data attributes start with data-. They are used to store extra information on an HTML element that doesn't have any visual impact. For example, <div data-user-id="123"> stores the user ID as a custom attribute. These attributes can be accessed via JavaScript using the dataset property.

17. How do you use the <picture> element for responsive images?

The <picture> element allows you to specify multiple image sources for responsive design. You can provide different image formats or resolutions that are used based on media conditions, such as screen size or pixel density. Inside the <picture> tag, you use <source> elements with a srcset attribute to define different image resources. For example:

<picture>
      <source media="(min-width: 800px)" srcset="large.jpg">
      <source media="(min-width: 400px)" srcset="medium.jpg">
      <img src="small.jpg" alt="Responsive image">
    </picture>

18. What is the purpose of the rel attribute in the <link> tag?

The rel attribute in the <link> tag specifies the relationship between the current document and the linked resource. For example, rel="stylesheet" tells the browser that the linked file is a CSS stylesheet. There are other values for rel, like rel="icon" for linking favicons and rel="nofollow" for links you don't want search engines to follow.

19. How do you use the srcset attribute in the <img> tag for responsive images?

The srcset attribute in the <img> tag is used to define a list of image sources with varying resolutions or sizes. The browser selects the appropriate image based on the user's device screen size and resolution. For example:

<img src="small.jpg" 
      srcset="medium.jpg 768w, large.jpg 1280w" 
      sizes="(max-width: 600px) 480px, 800px" alt="Responsive image">

20. What is the Shadow DOM and how can it be used in HTML?

The Shadow DOM is a web standard that allows you to encapsulate a section of the DOM tree in a separate "shadow" root. This is useful for building reusable components where the internal structure is isolated from the main document's styles and scripts. For example, custom elements can use shadow DOM to prevent style leakage. To create a shadow DOM, use JavaScript:

let shadow = element.attachShadow({ mode: 'open' });
    shadow.innerHTML = '<p>This is shadow DOM content</p>';

21. How do you implement ARIA roles and attributes for accessibility in HTML?

ARIA (Accessible Rich Internet Applications) roles and attributes improve web accessibility for users with disabilities. ARIA roles, such as role="button" or role="navigation", define the purpose of elements for screen readers. ARIA attributes like aria-label and aria-hidden provide additional context or hide elements from assistive technologies. Example: 

<button aria-label="Close window">X</button>

22. How can you optimize HTML performance for faster page load times?

To optimize HTML performance, you can follow best practices like minimizing the use of inline styles, reducing the number of HTTP requests, compressing images, using async or defer attributes for JavaScript, and using external stylesheets. Enabling caching and using a content delivery network (CDN) can also help speed up page load times.

23. What is the difference between <article>, <section>, and <aside> elements in HTML5?

The <article> element represents a self-contained composition that could be distributed independently, such as a blog post or news article. The <section> element defines a thematic grouping of content within a document. The <aside> element is used for content that is tangentially related to the main content, such as sidebars or related links.

24. How do you use the <template> element in HTML?

The <template> element is used to store HTML fragments that are not rendered when the page initially loads. The content inside the <template> can be cloned and inserted into the document later using JavaScript. This is useful for creating dynamic content. Example:

<template id="my-template">
      <div>This content will be rendered later.</div>
    </template>

25. What are the best practices for structuring an HTML5 document?

Best practices for structuring an HTML5 document include using semantic HTML tags like <header>, <footer>, <article>, and <section> to define meaningful sections of a page. Always include a DOCTYPE declaration, and use <meta> tags for character encoding and viewport settings to ensure proper display on all devices.

26. What is the purpose of the <!DOCTYPE html> declaration?

The <!DOCTYPE html> declaration tells the browser that the document is written in HTML5. It ensures that the browser renders the page in standards mode, which avoids quirks mode where old or incorrect HTML rendering might occur.

27. How do you add comments in HTML?

Comments in HTML are added using <!-- and -->. Anything written between these tags will not be rendered on the webpage. Comments are useful for annotating code and leaving notes for yourself or other developers. Example:

<!-- This is a comment -->

28. What are the differences between block-level and inline-level elements?

Block-level elements, like <div>, <p>, and <h1>, take up the full width of their container and always start on a new line. Inline-level elements, like <span> and <a>, take up only as much width as necessary and do not start on a new line. Block elements can contain inline elements, but inline elements cannot contain block elements.

29. What is the purpose of the <meta> tag in HTML?

The <meta> tag is used to provide metadata about the HTML document. Metadata is information about the data itself and is not displayed on the page. Common uses include specifying the character set (e.g., UTF-8), the viewport for responsive design, and descriptions for search engine optimization (SEO). Example:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<meta name="description" content="This is a description of the page.">

30. What is semantic HTML, and why is it important?

Semantic HTML refers to using HTML elements according to their intended meaning rather than for their appearance. For example, using <article> for content that could be distributed independently, rather than a <div>. This enhances accessibility, SEO, and code readability, making it easier for browsers and assistive technologies to interpret the content.

31. What is the difference between <ol> and <ul> elements in HTML?

The <ol> element creates an ordered list, where items are numbered automatically. The <ul> element creates an unordered list, where items are usually marked with bullet points. Both elements use <li> tags to represent list items. Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

32. What is the purpose of the <head> section in an HTML document?

The <head> section contains metadata and links to resources that the browser uses to render the page, but it is not visible to users. It usually includes elements such as <meta> tags, <title>, stylesheets, and scripts. The content of the <head> is essential for performance, SEO, and usability.

33. How do you create a table in HTML?

A table in HTML is created using the <table> element, with rows defined by <tr>, and data cells defined by <td>. Table headers use <th>. Example: 

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

34. What is the purpose of the <iframe> tag in HTML?

The <iframe> tag is used to embed another HTML document inside the current document. This is often used for embedding external content, such as videos or maps, from other websites. Example:

<iframe src="https://example.com" width="600" height="400">
  <p>Your browser does not support iframes.</p>
</iframe>

35. How do you create a hyperlink in HTML?

A hyperlink is created using the <a> tag. The href attribute specifies the URL of the page the link points to. The link text is placed between the opening and closing <a> tags. Example:

<a href="https://example.com">Visit Example.com</a>
